Списание товаров со склада.

Описание.

Процедура списания позволяет собрать неликвидный, повреждённый или просроченный товар со стеллажей при помощи терминала сбора данных.

Создание документа:

1) Нажимаем кнопку «Операции», далее выбираем строку «Списание».

2) Откроется окно создания документа «Списание», здесь проставляем:

  • Транзакцию – 301 «Списание доступного количества
  • Основание – «Изъятие товара из ячеек
  • Комментарий – указываем кратко назначение списание, число и Ф.И.О. исполнителя.

3) Сохраняем введенные значения.

4) Нажимаем на кнопку «Создать приказ для ввода строк», после этого в «Диспетчере» появится задание на списание, а кнопка изменит своё название на «Удалить приказ для ввода строк»(при нажатии на неё задание будет удалено из диспетчера).

5) Переходим в диспетчер и отправляем созданное задание по списанию на терминал. На терминале приказ будет выглядеть следующим образом:

6) Списание производится следующим образом:

  • Сканируем ячейку, откуда необходимо списать товар.
  • Сканируем сам товар.
  • Откроется таблица, в которой будет отображено количество товара и его сроки годности.
  • Двойное нажатие на строчку таблицы откроет окно редактирования количества. Это делаем в том случае, если списать нужно не весь срок годности, только часть.
  • Для списания нажимаем один раз на строку – она подсветится зелёным цветом.
  • После корректировки количества и выбора срока годности нажимаем «Сохранить»
  • Помним, что за одно сканирование ячейки можно списать лишь одну строку (при выделении нескольких строк появится сообщение об ошибке).

7) Программа вернётся к изначальной форме, но уже с заполненными строками. Нажимаем кнопку «Подтвердить», после этого данный товар спишется с ячейки и в документе списание добавится соответствующая строка. При нажатии на кнопку «Подтвердить» появится окно с подтверждением действия. Его следует внимательно прочитать перед нажатием «ОК».

8) После подтверждения предыдущего списания, строки таблицы очистятся и можно будет сканировать следующую ячейку и товар.

9) Для завершения работы с документом надо нажать кнопку «Меню» и выбрать строку «Завершить списание».

Списание в документе.

Чтобы посмотреть списанные в рамках одного документа товары можно воспользоваться печатной формой документа..

Просмотр списания за период.

1) Чтобы посмотреть все товары, которые были списаны за промежуток времени можно воспользоваться отчётом «Журнал списаний». Найти его можно здесь:

2) Откроется окно, где необходимо выбрать «Период отчёта» и «Основание» (Изъятие товара из ячеек). После этого нажимаем кнопку «Отчёт».

3) Через несколько секунд будет сформирован отчёт «Журнал списаний» за выбранный промежуток времени и по выбранному основанию.
